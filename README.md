# sjaimes11.github.io

Portafolio personal construido con React, TypeScript y Vite.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Estructura

- `src/data/portfolio.ts`: contenido editable del sitio
- `src/sections/*`: secciones principales
- `src/components/*`: componentes reutilizables
- `src/styles/global.css`: sistema visual y layout

## Despliegue

El repo incluye workflow de GitHub Pages en `.github/workflows/deploy.yml`.
